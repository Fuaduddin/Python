**Using if-else**
```python
number1 = int(input())
number2 = int(input())
if number1 > number2:
    print(number1, "is maximum")
else:
    print(number2, "is maximum")
```

**Using built-in Function**
```python
number1 = int(input())
number2 = int(input())
maximum = max(number1, number2)
print(maximum, "is maximum")
```
